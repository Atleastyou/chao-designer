# 巢搭配店铺装修组件

## 安装依赖
```
npm install
```

### 运行
```
npm run dev
```

### 编译
```
npm run lib
```

### 登录
```
npm login
```

### 发布
```
npm publish
```


## 目录结构介绍 ##

	|-- examples                         // demo
	|-- lib                              // 编译后的代码
	|-- packages                         // 组件
	|-- public                           // 入口模板
	|-- utils                            // 工具JS
	|-- .gitignore                       // git忽略文件
	|-- vue.config.js                    // vue cli 配置
	|-- README.md                        // 说明
